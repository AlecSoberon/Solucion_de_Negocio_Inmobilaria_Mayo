Portafolio web del proyecto Villa Verde para Inmobiliaria Mayo. Presenta el caso de negocio, la arquitectura analítica, el modelo de datos, el dashboard en Power BI y las oportunidades de automatización para mejorar la gestión comercial.

Resumen ejecutivo

El proyecto busca convertir datos comerciales dispersos en una herramienta de decisión para monitorear el ratio de conversión, identificar cuellos de botella y priorizar acciones de mejora en captación, calificación y gestión de leads.
