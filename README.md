# Grupo 05 · Inmobiliaria Mayo

Portafolio web del proyecto de análisis del ratio de conversión comercial para Inmobiliaria Mayo, desarrollado para el curso Business Analytics 2026-Enero de la PUCP.

## Objetivo

Documentar la solución analítica del proyecto Villa Verde, incluyendo el caso de negocio, modelo de datos, arquitectura propuesta, dashboard en Power BI y oportunidades de automatización.

## Estructura

```text
index.html
pages/
  caso-negocio.html
  arquitectura.html
  data-layer.html
  etl-fabric.html
  powerapps.html
  powerbi.html
  automatizacion.html
  equipo.html
assets/
  css/styles.css
  js/main.js
  img/
```

## Publicación

El sitio se publica con GitHub Pages desde la rama `main`.
